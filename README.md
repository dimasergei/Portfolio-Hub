# Portfolio Hub

A high-end, responsive portfolio showcasing 5 production AI systems.

## Features

- **Responsive Design**: Built with Next.js, Tailwind CSS, and shadcn/ui
- **System Health Dashboard**: Real-time status of 5 production systems
- **Wake-up Script**: Automatically pings backend URLs to keep free-tier instances warm
- **Modern UI**: Glass morphism, gradients, and smooth animations
- **Performance Metrics**: Live performance stats for each project

## Projects

1. **EnterpriseRAG** - Production document Q&A with hybrid retrieval
2. **CodeScan AI** - Intelligent code analysis with Tree-sitter AST
3. **StreamPulse** - Real-time event streaming with anomaly detection
4. **AgentIQ** - Multi-agent system with visible thought process
5. **MeetingGPT** - AI-powered meeting transcription with action items

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion
- **Infrastructure**: Vercel (hosting)

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel with automatic wake-up functionality for free-tier backend services.
